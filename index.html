Relational

x == y

- validates if both are equal

x != y

- validates if both are different

x > y

- validates if "x" is higher than "y"

x >= y

- validates if "x" is higher or equal to "y"

x < y

- validates if "x" is lower than "y"

x <= y

- validates if "x" is lower or equal to "y"

x === y

- validates if both are identical (same values and type)

x !== y

- validates if both are not identical (different values and type)

Logical

test || test

Ternary

- executes the True or the False path depending on the test's result

:[...] it's the False path
